Grocery Project

A grocery website where users can fill their carts with all the necessary shopping items.

Everything is categorized.

Two types of users: Admin and customer, where each have different webpages.


Technologies used:
ASP.NET Web Forms - C#

I've used visual studio as an IDE, in addition to C# as a programming language for the backend.

SQL

Our website is linked to a database that stores all users' information in a secured way.

    What a customer can do:
  • Add items to cart
  • Make orders
  • Submit reports

    What an admin can do:
  • Add items to be visible on customers' end
  • Add categories to be visible on customers' end
  • Check reports
  • Deliver orders
